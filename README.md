# OpenFGA (openfga)
OpenFGA is a CNCF incubating fine-grained authorization system based on the Zanzibar model. It provides relationship-based access control through HTTP and gRPC APIs with a flexible type system.

## APIs

### OpenFGA Authorization API
HTTP and gRPC endpoints for Check, Write, Read, ListObjects, and Expand operations supporting fine-grained authorization with relationship tuples and model versioning.
